\CWProblem{
Необходимо реализовать один из стандартных алгоритмов поиска образцов для указанного алфавита.

{\bfseries Вариант алгоритма:} Поиск одного образца при помощи алгоритма Апостолико-Джанкарло.

{\bfseries Вариант алфавита:} Числа в диапазоне от \(0\) до \(2^{32} - 1\)
}
